# Hi, I'm dev-Tofu-hello 👋

## About Me
インフラからアプリケーション層まで横断して開発することが好きです。

自宅にProxmox VEベースの仮想化基盤を構築し、
セルフホストVPNによるメッシュネットワーク、
CloudflareのOrigin Rulesを用いたトラフィックフォワーディング、
MagicDNSによる内部名前解決、Nginxによるリバースプロキシ設計など、
ネットワーク層から一貫したインフラ設計・運用を得意としています。

「なぜ動かないか」を根本から追う泥臭いトラブルシューティングと、
新しい技術を実際に手を動かして検証することが好きです。

バックエンド開発にも対応しており、Python・Javaでの
実装経験があります。現在はC#の学習も進めています。
